# SODA
# 上海开放数据创新应用大赛商圈公共安全智能app
 - 源文件包含：
   - Test_Soda
   - Test_SodaTests
   - Test_SodaUITests
   - Pods
   - aws-ios-sdk-2.4.10
   - opencv2.framework
   - podfile
   - podfile.lock
   - Test_Sodacopy.xcworkspace（project文件）
   - Test_Soda.xcodeproj
   - Test_Soda.scworkspace
   
- 使用说明：
 - 使用Mac电脑下载Xcode8
 - 打开Test_Sodacopy.xcworkspace
 - 选择iPad Air 2作为simulator，点击运行键
 - 目前可以点击的按钮：
 
   - 第一页有“上海地铁”，可转至上海地铁官网
   
   - 第一页有“南京东路商圈”；
    - 第二页有“17：00 - 18：00”， “18：00 - 19：00”和“19：00 - 20：00”
